图片预览视图

# 属性
| 属性 | 含义 |
| ---- | ---- |
| needKeepScale | 是否需要图片与视图比例一样|
| shouldCropOut | 超出部分是否需要裁剪掉 |
| minScale | 最小比例, 默认0.95 |
| maxScale | 最大比例, 默认3.0 |
| image | 内容图片 |
